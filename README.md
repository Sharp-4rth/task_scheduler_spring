# Task Scheduler API

## Overview
Simple Spring Boot REST API for managing tasks.

## Logic
- Tasks are assigned a basic priority on creation (simple greedy approach).

## Tech Stack
- Java (Maven, JDK 21)
- Spring Boot
- H2 Database

## Endpoints
- POST /tasks
- GET /tasks
- GET /tasks/{id}

## How to Run
- Run main class
- Access H2 console
- Use Postman to test endpoints
